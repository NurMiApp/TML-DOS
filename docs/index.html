Terminal Commands Guide

Package Management Commands

APT (Debian-based systems)

Install a package: sudo apt install package_name

Update package list: sudo apt update

PKG (FreeBSD systems)

Install a package: pkg install package_name

Using Nano Text Editor

Edit a file: nano filename

File System Operations

List Directory Contents

ls - Lists directory contents

ls -l - Lists in long format with more details

Change Directory

cd /path/to/directory - Change to specified directory

cd .. - Move up one directory

Create a Directory

mkdir new_directory_name - Create a new directory

Remove Files or Directories

rm file_name - Remove a file

rm -r directory_name - Remove a directory and its contents recursively
